📌 系統概覽

JSONL Timeline Visualizer 是一個專為 Claude 對話記錄設計的視覺化工具。它能夠將 JSONL 格式的對話記錄轉換成直觀的時間軸視圖，幫助您更好地理解和分析 AI 對話流程。

💡 核心概念：
  • 左側（本地）：顯示在您電腦上執行的操作，如使用者輸入、檔案讀寫等
  • 右側（雲端）：顯示在伺服器執行的操作，如 AI 回應、網路搜尋等
  • 中線時間軸：串連所有事件，按時間順序排列

🚀 快速開始

步驟 1：開啟應用程式

npm run dev # 或者 npm start

步驟 2：載入 JSONL 檔案

點擊右上角的「Upload JSONL」按鈕，選擇您的對話記錄檔案。

上傳介面示意圖 - 點擊按鈕後會出現檔案選擇對話框

步驟 3：探索時間軸

檔案載入後，您將看到完整的對話時間軸。您可以：

  • 滾動瀏覽所有事件
  • 點擊展開詳細內容
  • 使用搜尋功能快速定位
  • 匯出您需要的內容

🖥️ 介面說明

頂部導航欄

JSONL Timeline Visualizer
導航欄包含：標題、搜尋框、上傳按鈕、匯出按鈕、深色模式切換

統計資訊區

顯示當前載入檔案的統計資訊：

  • Total Events：總事件數
  • User Messages：使用者訊息數
  • Assistant Messages：AI 回應數
  • Tool Calls：工具呼叫次數

事件類型說明

使用者訊息

藍色邊框，顯示使用者輸入的內容

AI 回應

綠色邊框，顯示 AI 的回覆內容

工具請求

紫色邊框，顯示工具呼叫請求

工具結果

橙色邊框，顯示工具執行結果

⏱️ 時間軸功能

展開與折疊

點擊事件卡片右上角的箭頭圖標，可以展開或折疊相關的子事件：

提示：工具請求和結果通常是成對出現的，展開工具請求可以看到對應的執行結果。

時間戳記

每個事件都顯示精確的時間戳記，格式為 HH:mm:ss.SSS，方便追蹤事件發生的順序。

🔍 搜尋功能

基本搜尋

  1. 在頂部搜尋框輸入關鍵字
  2. 系統會即時高亮顯示所有匹配的內容
  3. 匹配的文字會以黃色背景標示
  4. 點擊 X 或按 ESC 鍵清除搜尋

搜尋技巧

💡 專業提示：
  • 搜尋不區分大小寫
  • 可以搜尋中文、英文、數字
  • 特殊字符會被當作普通文字處理
  • 搜尋會包含所有事件內容，包括工具參數

🌙 深色模式

深色模式可以減少眼睛疲勞，特別適合長時間使用或在光線較暗的環境中工作。

切換方式

點擊右上角的太陽/月亮圖標即可切換明暗主題。您的偏好設定會自動儲存在瀏覽器中。

注意：深色模式會同時影響所有介面元素，包括背景、文字、邊框等。如果您發現任何顯示問題，請嘗試重新整理頁面。

最佳實踐

  1. 從折疊視圖開始：初次載入時保持事件折疊，只展開需要查看的部分
  2. 善用搜尋功能：快速定位特定對話或工具呼叫
  3. 定期匯出：重要的對話記錄及時匯出備份
  4. 注意檔案大小：大型檔案可能影響效能，建議分割處理
  5. 使用深色模式：長時間使用時切換到深色模式保護眼睛

🔧 疑難排解

常見問題

❌ 檔案無法載入

  • 確認檔案格式為有效的 JSONL
  • 檢查檔案大小（建議小於 50MB）
  • 查看瀏覽器控制台是否有錯誤訊息

❌ 時間軸顯示異常

  • 重新整理頁面清除快取狀態
  • 將瀏覽器縮放比例重設為 100%
  • 嘗試使用其他瀏覽器

❌ 搜尋功能無回應

  • 清空搜尋框重新輸入
  • 檢查是否包含特殊字符
  • 如果卡住，重新載入頁面

❌ 匯出失敗

  • 檢查瀏覽器是否阻擋彈出視窗
  • 確認磁碟空間充足
  • 嘗試其他匯出格式
💡 提示：如果問題持續存在，請嘗試：
  1. 清除瀏覽器快取和 Cookie
  2. 使用無痕/隱私模式
  3. 更新瀏覽器到最新版本
  4. 聯繫技術支援團隊
